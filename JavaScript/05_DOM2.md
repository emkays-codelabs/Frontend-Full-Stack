With DOM manipulation, you can:

# 🔹 2. Working with Attributes (IMPORTANT)

---

## ✅ Get Attribute

```html
<img id="img" src="photo.jpg" alt="image">
```

```js
const img = document.getElementById("img");

console.log(img.getAttribute("src"));  // photo.jpg
```

---

## ✅ Set Attribute

```js
img.setAttribute("src", "new.jpg");
```

👉 Replaces existing value OR adds new attribute

---

## ✅ Remove Attribute

```js
img.removeAttribute("alt");
```

---

## 🔥 Shortcut (Property Access)

```js
img.src = "updated.jpg";
img.alt = "new image";
```

👉 Faster and cleaner than `setAttribute`

# 🔹 3. Changing Styles

---

## ❌ Direct Style (Not Best)

```js

el.style.backgroundColor = "black";
el.style.fontSize = "20px";
```

---

## ✅ Best Way: classList

```css
.active {

  background: black;
}
```

```js
el.classList.add("active");
```

---

## 🔥 classList Methods

```js

el.classList.remove("class1");
el.classList.toggle("class1");
el.classList.contains("class1");
```

# 🔹 5. Inserting Elements (VERY IMPORTANT 🔥)

---

## 🧱 Base HTML

```html
<div id="parent">
  <p>Existing</p>
</div>
```

---

## ✅ append() (Modern)

```js
parent.append(div);
```

👉 Adds at **end**
👉 Can add multiple elements
👉 Can add text also

---

## ✅ prepend()

```js
parent.prepend(div);
```

👉 Adds at **start**

---

## ✅ appendChild() (Older but Important)

```js
parent.appendChild(div);
```

👉 Only accepts **Node (element)**
👉 Returns inserted element

---

## 🔥 Difference

| Method      | Accepts Text | Multiple | Modern |
| ----------- | ------------ | -------- | ------ |
| append      | ✅            | ✅        | ✅      |
| appendChild | ❌            | ❌        | ❌      |

---

## ✅ before() / after()

```js
const existing = document.querySelector("p");

existing.before(div);  // before p
existing.after(div);   // after p
```

---

## ✅ insertAdjacentHTML (Advanced)

```js
parent.insertAdjacentHTML("beforeend", "<p>New</p>");
```

### Positions:

* `"beforebegin"`
* `"afterbegin"`
* `"beforeend"`
* `"afterend"`

# 🔹 9. Working with Multiple Elements

```js

for (let i = 1; i <= 5; i++) {
  const div = document.createElement("div");
```

```js
});
```

# 🔹 10. Real-World Example (FULL)

---

## HTML

```html
<div id="list"></div>
<button id="add">Add Item</button>
```

---

## JS

```js
const list = document.getElementById("list");
const btn = document.getElementById("add");

let count = 1;

btn.addEventListener("click", () => {
  const div = document.createElement("div");

  div.textContent = "Item " + count;
  div.classList.add("item");

  // Add remove button inside
  const removeBtn = document.createElement("button");
  removeBtn.textContent = "Delete";

  removeBtn.addEventListener("click", () => {
    div.remove();
  });

  div.append(removeBtn);
  list.append(div);

  count++;
});
```
Perfect — now let’s do this the **right way (clear + simple + with examples)** so you can *actually master DOM manipulation*.

# 🔥 INSERT OPTIONS (With Code Examples)

---

## ➤ 1. append() → add at END (inside element)

### 🧠 Concept:

Adds content **as the last child inside a parent**

### ✅ Example:

```html
<div id="parent"></div>
```

```js
const parent = document.getElementById("parent");

const newEl = document.createElement("p");
newEl.textContent = "I am appended";

parent.append(newEl);
```

👉 Result:

```html
<div id="parent">
  <p>I am appended</p>
</div>
```

---

## ➤ 2. prepend() → add at START

### 🧠 Concept:

Adds content **as the first child**

```js
parent.prepend(newEl);
```

👉 Result:

```html
<div id="parent">
  <p>I am first</p>
  ...
</div>
```

---

## ➤ 3. before() → before element (outside)

### 🧠 Concept:

Insert **before a specific element (same level)**

```html
<p id="target">Target</p>
```

```js
const target = document.getElementById("target");

const el = document.createElement("p");
el.textContent = "Before element";

target.before(el);
```

👉 Result:

```html
<p>Before element</p>
<p id="target">Target</p>
```

---

## ➤ 4. after() → after element (outside)

```js
target.after(el);
```

👉 Result:

```html
<p id="target">Target</p>
<p>After element</p>
```

---

## 🧠 Summary

| Method  | Position         |
| ------- | ---------------- |
| append  | End              |
| prepend | Start            |
| before  | Outside (before) |
| after   | Outside (after)  |

---

# 🔥 REMOVE OPTIONS (With Code Examples)

---

## ➤ 1. remove() → easiest

### 🧠 Concept:

Element removes itself

```html
<p id="removeMe">Delete me</p>
```

```js
const el = document.getElementById("removeMe");
el.remove();
```

---

## ➤ 2. removeChild() → parent removes child

### 🧠 Concept:

Parent deletes its child

```html
<div id="parent">
  <p id="child">Child</p>
</div>
```

```js
const parent = document.getElementById("parent");
const child = document.getElementById("child");

parent.removeChild(child);
```

---

## 🧠 Difference

| Method      | Needs Parent |
| ----------- | ------------ |
| remove      | ❌            |
| removeChild | ✅            |

# 🚀 PRACTICE PROJECTS (With Idea + Code Skeleton)

---

## 🔹 1. Dynamic List (Add/Delete)

```js
const ul = document.querySelector("ul");

function addItem(text) {
  const li = document.createElement("li");
  li.textContent = text;

  const btn = document.createElement("button");
  btn.textContent = "Delete";

  btn.onclick = () => li.remove();

  li.append(btn);
  ul.append(li);
}
```

---

## 🔹 2. Image Gallery (change src)

```js
const img = document.querySelector("img");

function changeImage(newSrc) {
  img.setAttribute("src", newSrc);
}
```

---

## 🔹 3. Card Generator

```js
const container = document.getElementById("cards");

function createCard(title) {
  const card = document.createElement("div");

  container.append(card);
}
```

## 🔹 4. Form Builder

```js
form.addEventListener("submit", (e) => {

  document.body.append(p);
});
```

## 🔹 5. Notification System

```js
function showNotification(msg) {
  const div = document.createElement("div");

  div.textContent = msg;
  div.classList.add("notification");

  document.body.append(div);

  setTimeout(() => {
    div.remove();
  }, 2000);
}
```
